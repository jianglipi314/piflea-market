Piflea 用户协议与费用说明

版本：v1.0  ·  生效日期：2026年6月20日

适用平台：Piflea（piflea.com）—— Pi 生态二手跳蚤市场

重要声明

  1. Piflea 是独立运营的第三方平台，与 Pi Network、Pi Core Team 及其关联公司无任何官方隶属、合作或授权关系。
  2. Piflea 仅提供信息展示与交易撮合服务，不直接参与买卖双方的交易，也不对商品质量做担保（担保交易服务除外）。
  3. 使用 Piflea 即表示您已阅读、理解并同意本协议全部内容。如您不同意，请立即停止使用。

一、定义与解释

术语含义
Piflea / 平台指由 Piflea 团队运营的二手交易信息服务平台
用户在 Piflea 注册并使用服务的个人，包括买家和卖家
卖家在平台上发布商品出售信息的用户
买家在平台上浏览并购买商品的用户

担保交易指买家支付的 Pi 由平台托管，确认收货后再释放给卖家的服务
区块链网络费指 Pi 区块链网络收取的固定交易确认费用（目前为 0.01 π/笔）

二、账户与身份验证

2.1 注册要求

  • 您必须使用有效的 Pi Network 账号登录 Piflea。
  • 根据 Pi 主网要求，买卖双方均需完成 Pi KYC 身份验证方可进行交易。
  • 您保证提供的身份信息真实、准确、有效。

2.2 账户安全

  • 您应妥善保管自己的 Pi 钱包私钥和账户信息。
  • 因您泄露账户信息导致的损失，平台不承担责任。

  • 平台保留在未来收取合理服务费的权利。
  • 任何费率调整将提前 30 天在平台公告，不溯及已发生的交易。

  • 区块链网络费由 Pi 区块链网络收取，非 Piflea 收取。

  • 网络费标准由 Pi 网络协议决定，平台无权调整。

四、交易规则

4.1 卖家规则

  1. 真实描述：商品信息必须真实、准确，包括但不限于商品成色、功能、瑕疵、使用年限等。
  2. 合法商品：严禁发布违禁品（详见第 7 条）。
  3. 及时发货：买家付款后，卖家应在承诺时间内发货或完成交付。
  4. 如实定价：价格应合理反映商品价值，禁止恶意炒作或欺诈性定价。

4.2 买家规则

  1. 谨慎购买：二手商品存在固有瑕疵，请仔细阅读商品描述并与卖家充分沟通。
  2. 及时确认：收到商品并验收无误后，应在 7 天内确认收货。
  3. 诚信交易：禁止恶意下单、恶意退款或利用平台漏洞谋取不当利益。

4.3 交易流程

买家下单支付 → 资金进入担保托管（如选择担保交易）
    ↓
卖家发货/交付 → 提供交付凭证
    ↓
买家验收 → 确认收货或申请退款
    ↓
确认收货：资金释放给卖家  |  申请退款：进入争议处理流程

五、担保交易服务

5.1 服务说明

  • Piflea 提供可选的担保交易服务。
  • 选择担保交易后，买家支付的 Pi 将由平台托管，不会立即转给卖家。
  • 买家确认收货后，平台将在 24 小时内将资金释放给卖家。

5.2 担保期限

  • 默认担保期为 7 天（从卖家标记发货/交付完成起算）。
  • 担保期内买家可申请退款；逾期未操作，系统自动确认收货。

5.3 退款规则

  • 商品与描述严重不符
  • 商品存在功能性瑕疵且卖家未提前说明
  • 卖家未按约定发货/交付
  • 其他经平台认定应退款的情形

注意：因二手商品特殊性，轻微磨损、外观瑕疵如已在描述中说明，不支持退款。

六、风险提示

6.1 二手交易固有风险

  • 二手商品可能存在隐藏瑕疵、老化、功能衰退等问题。
  • 平台仅提供信息展示，不对商品质量、真伪、合法性做担保。
  • 建议买家通过担保交易降低风险，收货时仔细验收。

6.2 区块链交易风险

  • Pi 区块链交易不可逆转，一旦确认无法撤销。
  • 网络拥堵、节点故障等可能导致交易延迟，平台不承担责任。
  • 请确保您的 Pi 钱包地址正确，转错地址无法找回。

6.3 价格波动风险

Pi 的市场价格可能存在波动，商品定价以 Pi 计价，法币价值可能变化。交易双方应自行承担价格波动风险。

6.4 平台服务风险

Piflea 处于早期运营阶段，可能出现技术故障、服务中断等情况。平台将尽力保障服务稳定，但不对不可抗力（如战争、自然灾害、网络攻击）导致的损失负责。

七、禁止行为与违禁品

7.1 严禁发布的商品

  • 毒品、枪支、弹药、爆炸物等违禁品
  • 伪造货币、证件、票据
  • 盗窃物、赃物
  • 侵犯知识产权的商品（盗版、假冒品牌）
  • 色情、赌博相关物品
  • 活体动物（部分国家/地区法律禁止）
  • 其他违反当地法律法规的商品

7.2 严禁的行为

  • 洗钱、恐怖融资等非法金融活动
  • 利用平台进行诈骗、钓鱼、诱导线下交易绕过担保
  • 发布虚假信息、恶意刷单、操纵评价
  • 侵犯他人隐私、骚扰、威胁其他用户
  • 攻击平台系统、利用漏洞谋取不当利益

违规后果：平台有权下架商品、冻结账户、封禁账号，并配合有关部门调查。

八、知识产权

  • 用户在平台发布的商品图片、描述等内容的知识产权归用户所有。
  • 用户授权平台在运营范围内使用上述内容（如展示、推广）。
  • 禁止未经授权使用他人图片、文字，侵权者自行承担法律责任。

九、隐私保护

  • 平台仅收集交易必需的信息（如 Pi 用户名、KYC 状态、交易记录）。
  • 平台不会向第三方出售用户数据。
  • 为完成交易和争议处理，平台可能需要向交易对方披露必要信息。
  • 详细隐私政策请参见《Piflea 隐私政策》。

十、免责声明

  1. 平台性质：Piflea 是信息撮合平台，非商品卖方或买方，不对交易商品的质量、安全、合法性负责。
  2. 用户行为：因用户自身行为（如泄露密码、转错地址、线下交易被骗）导致的损失，平台不承担责任。
  3. 第三方服务：因 Pi Network、区块链网络、第三方服务商（如地图、支付接口）故障导致的损失，平台不承担责任。
  4. 信息真实性：平台尽力审核信息，但无法保证所有商品信息 100% 真实，用户应自行判断。

十一、争议处理

11.1 协商优先

  • 交易纠纷首先由买卖双方协商解决。
  • 平台提供沟通渠道，鼓励友好协商。

11.2 平台介入

  • 协商不成，任何一方可在担保期内申请平台介入。
  • 平台将根据双方提供的证据（聊天记录、发货凭证、商品照片等）做出裁决。
  • 平台裁决为最终结果，双方应遵守。

11.3 证据要求

  • 卖家：发货凭证、商品描述截图、沟通记录
  • 买家：收货照片、商品瑕疵照片、沟通记录
  • 证据不足的一方承担不利后果。

十二、协议变更与终止

  • 平台有权根据运营需要修改本协议，修改后将在平台公告。
  • 如您不同意修改内容，应停止使用平台服务。
  • 如您严重违反本协议，平台有权终止服务、封禁账户。
